El código html { font-size: 62.5%; } establece el tamaño de la fuente en el 62,5% del valor predeterminado del elemento raíz. Esto es útil porque simplifica las cosas, y el uso de valores de longitud relativa como REM es importante para la accesibilidad. Así es como funcionan los tamaños de fuente en este ejemplo: html { tamaño de fuente: 62.5%; } Establece el tamaño de fuente en el 62,5% del valor predeterminado del elemento raíz, que es 10px. h1 { tamaño de fuente: 1.8rem; } Establece el tamaño de fuente del elemento h1 en 1,8 veces el valor de 10px, que es 18px. De forma predeterminada, los navegadores establecen el tamaño de fuente HTML en 16px, por lo que 1rem (tamaño de fuente del elemento raíz) es igual a 16px. Sin embargo, los usuarios pueden cambiar este valor predeterminado en la configuración de su navegador o sistema operativo.

html{
font-size: 62.5%;
}
h1{
font-size: 3rem;
}
p{
font-size: 1.8rem;
}
