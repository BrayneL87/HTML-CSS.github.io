sibling

En CSS, sibling se refiere a los selectores de hermanos, que son símbolos que ayudan a los navegadores a comprender la relación entre los elementos HTML.

h2 ~ p{
color: blue;
}

Combinador Hijos y decendiente

En CSS, el selector de descendientes para seleccionar todos los elementos p dentro de un elemento div es div p. El selector de hijo directo es div > p. Este selector solo coincide con los elementos que son hijos directos de los elementos coincidentes con el primer selector.

div > p{
color: red;
}
