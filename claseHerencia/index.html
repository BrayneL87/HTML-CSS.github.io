Tipo de letra

font-family:

se utiliza para especificar el tipo de letra o fuente que se mostrará en un elemento. Es decir, define la apariencia visual del texto dentro de un elemento HTML.

se crea una etiqueta html en css y utiliza font-family para agregar el tipo de letras qie desees

html{

font-size: 75%;

font-family: Verdana, Geneva, Tahoma, sans-serif;

}


font-size: inherit;

es una herramienta muy útil para heredar el tamaño de fuente de un elemento padre a sus elementos hijos. Esto significa que el tamaño de la fuente de los elementos hijos se ajustará automáticamente al tamaño de fuente definido para su elemento padre.

h1{

font-size: inherit;

}
